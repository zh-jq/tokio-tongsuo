# tokio-tongsuo

An implementation of SSL streams for Tokio built on top of the [`tongsuo` crate]

[`tongsuo` crate]: https://github.com/zh-jq/rust-tongsuo

, based on [tokio-openssl](https://github.com/tokio-rs/tokio-openssl).
